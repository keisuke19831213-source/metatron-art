# METATRON — 立体メタトロン方陣

**Live:** https://metatron-art.vercel.app

3次元メタトロンキューブ(十三の球・七十八の光径)をテーマにした、インタラクティブなデジタルアート作品。
外部ライブラリを一切使わない純 WebGL2 のシングルファイル実装です。

## 表現技術

- **HDR レンダリングパイプライン** — 半精度浮動小数点フレームバッファ → ブライトパス → 3段階ダウンサンプル・ガウシアンブルーム → ACES フィルミックトーンマッピング
- **加算合成の光条** — 78本のエッジをスクリーンスペースで展開したクアッドとして描画(コア+ハローの二重減衰)
- **仕上げ** — 色収差・ビネット・フィルムグレイン・FBM星雲・視差つき星空・神聖幾何学の補助円
- **生成音響** — WebAudio によるアンビエントドローンとペンタトニックの鐘(音源ファイルなし)

## インタラクション

| 操作 | 効果 |
| --- | --- |
| ドラッグ | 慣性つき軌道回転 |
| ホイール / ピンチ | ズーム |
| クリック / タップ | 中心から波動が伝播 |
| 平面⇄立体スライダー | 2D メタトロンキューブ ⇄ ベクトル平衡体の連続モーフ |
| 自動運行 | 26秒周期で呼吸するように次元が変容 |
| 彩 | 黄金 / 蒼氷 / 紫苑 の3パレット切替 |
| 音 | 生成アンビエントの ON/OFF |

## 実行

`index.html` をブラウザで開くだけで動作します(WebGL2 必須)。ビルド・依存関係なし。
